Assignment 2

Create a function which will allow a user to change the background color of the div with an id of "box" below. When a user types a color into the input area below and clicks "Submit Color", the background color of the box should change to that color. This challenge can be done in 5 lines of code or less.

Tips

  • Assign the button to a variable
  • Set that button to a onclick function
  • AFTER the button is clicked, get the input value
  • Change the background color with the .style method
